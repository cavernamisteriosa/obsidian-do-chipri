# Visão Geral

A prisão de [[Kaerum]] onde [[Larriel Belier]] e o resto do [[Ultimo Vagão]] estavão presos. Ela se localiza dentro de uma das ilhas voadoras. Atualmente estão presos lá [[Brocolis]], [[Sukhoi]] e [[Jimbo]].

## Descrição

A prisão é dividida em camadas. Entre as camadas beta e omega há um laboratorio especial do núcleo.

- [[Camada Alfa]]
- [[Camada Beta]]
- [[Camada Omega]]
- [[Laboratorio d'O Núcleo]]

### Camada Alfa

Crimes leves.

### Camada Beta

Crimes intermediarios como terrorismo.

### Camada Omega

Camada para crimes hediondos.
